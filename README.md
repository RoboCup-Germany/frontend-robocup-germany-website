# frontend-robocup-germany-website
Frontend of the RoboCup Germany Website

First Steps:
1. install npm/yarn/pnpm (e.g. https://pnpm.io/installation)
2. go into /src folder (`cd frontend.../src`)
3. install packages once `pnpm i`
4. run frontend locally `pnpm dev`
5. ... profit?

Remote deployment is handled by dockerfile
